Inside the Box

The BlipBox consists of a few simple elements cleverly put together. At the core there is a 10x8 matrix of super-bright LED's with individually adjustable brightness. Overlaying this is a precision touch screen. On the side you will find some more controls; in the current incarnation there is one button and one knob.

The box plugs straight into a computer's USB port. Being bus-powered, it doesn't require batteries or power adaptor.

The current prototypes use an arduino board which has an ATMega168 microprocessor and USB connection. We are planning to provide DIY kits quite soon, allowing anyone to build their own box.
